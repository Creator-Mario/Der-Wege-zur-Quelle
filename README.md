# Die Bibel – Glauben der Weg zur Quelle

Eine Bibelstudium-Anwendung mit strukturierten Themen und Lektionen.

## Übersicht

Diese Anwendung bietet ein umfassendes Bibelstudium-Programm mit 6 Hauptthemen und insgesamt 30 Lektionen. Jede Lektion enthält:
- Eine detaillierte Beschreibung
- Relevante Bibelstellen
- Einen Kernvers zur Vertiefung

## Themen

1. **Die Grundlagen des Glaubens** (5 Lektionen)
2. **Der Weg zur Quelle - Leben mit Gott** (6 Lektionen)
3. **Das Leben als Christ** (5 Lektionen)
4. **Die Verheißungen Gottes** (5 Lektionen)
5. **Der Heilige Geist** (4 Lektionen)
6. **Die Bergpredigt** (5 Lektionen)

## Dokumentation

Eine vollständige Auflistung aller Themen und Lektionen finden Sie in der Datei [THEMEN_UND_LEKTIONEN.md](THEMEN_UND_LEKTIONEN.md).

## Ausführung

Um die Anwendung zu starten:

```bash
cd Bibelgemeinschaft
dotnet run
```

Die Anwendung zeigt eine übersichtliche Auflistung aller verfügbaren Themen und Lektionen mit ihren jeweiligen Bibelstellen und Kernversen an.
